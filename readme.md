## ✨ Features / Fitur Unggulan

- **🌗 Dark Futuristic UI:** Desain modern dengan efek *glow* dan *glassmorphism* yang elegan.
- **🌐 Bilingual Ready (ID/EN):** Fitur ganti bahasa otomatis tanpa *reload* menggunakan Native JavaScript.
- **📱 Mobile Optimized:** Navigasi *fullscreen overlay* dengan animasi *Hamburger-to-X* yang mulus.
- **⚡ Blazing Fast:** Tidak menggunakan *framework* berat. Murni HTML, Tailwind CSS (via CDN), dan Vanilla JS.
- **🎡 Infinite Marquee:** Animasi *tech-stack* berjalan tanpa henti untuk visual yang lebih dinamis.
- **🔍 SEO Friendly:** Struktur tag semantik dan meta data yang siap di-index Google.

---

## 🛠️ Tech Stack / Alat Tempur

- **Markup:** HTML5
- **Styling:** Tailwind CSS (CDN Configured)
- **Logic & Interactions:** Vanilla JavaScript
- **Icons:** [Lucide Icons](https://lucide.dev/)

## 🚀 Quick Start / Cara Penggunaan

Karena *template* ini menggunakan Tailwind via CDN, kamu **tidak perlu** menginstal Node.js atau NPM. Tinggal jalankan!

1. **Clone repository ini:**
   ```bash
   git clone [https://github.com/ziyad/futuristic-portfolio.git](https://github.com/ziyad/futuristic-portfolio.git)

2. **Buka foldernya:**

cd futuristic-portfolio

3. **Jalankan:**
Buka file Porto.html langsung di browser kamu, atau gunakan ekstensi Live Server di VS Code agar lebih nyaman.

🎨 Customization / Cara Modifikasi
Template ini sangat mudah disesuaikan dengan profil kamu sendiri:

1. Ganti Teks & Bahasa: Buka file Porto.html, scroll ke bagian paling bawah ke dalam <script>. Temukan object const translations = { ... }. Ganti teks bahasa Indonesia dan Inggris di sana sesuai profil kamu.

2. Ganti Link Sosial Media:
Cari tag href="https://github.com/ziyad" (Gunakan Ctrl+F atau Cmd+F) lalu ganti dengan username kamu.

3. Ganti Warna Utama:
Di bagian <head>, cari tailwind.config. Kamu bisa mengubah kode hex pada neonBlue dan neonPurple untuk mengganti tema warna keseluruhan.
