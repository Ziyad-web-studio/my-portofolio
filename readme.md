## ✨ Features / Fitur Unggulan

- **🌗 Dark Futuristic UI:** Desain modern dengan efek *glow* dan *glassmorphism* yang elegan.
- **🌐 Bilingual Ready (ID/EN):** Fitur ganti bahasa otomatis tanpa *reload* menggunakan Native JavaScript.
- **📱 Mobile Optimized:** Navigasi *fullscreen overlay* dengan animasi *Hamburger-to-X* yang mulus.
- **⚡ Blazing Fast:** Tidak menggunakan *framework* berat. Murni HTML, Tailwind CSS (via CDN), dan Vanilla JS.
- **🎡 Infinite Marquee:** Animasi *tech-stack* berjalan tanpa henti untuk visual yang lebih dinamis.
- **🔍 SEO Friendly:** Struktur tag semantik dan meta data yang siap di-index Google.

---

## 🛠️ Tech Stack / Alat Tempur

- **Markup:** HTML5
- **Styling:** Tailwind CSS (CDN Configured)
- **Logic & Interactions:** Vanilla JavaScript
- **Icons:** [Lucide Icons](https://lucide.dev/)
